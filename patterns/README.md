# Pattern Library

Reusable local patterns for future agent work. Use this folder before inventing a new approach for a similar 3D/frontend task.

## Index

- `threejs-scroll-card-rail.md`: camera rail that moves from card to card on scroll.
- `wet-iridescent-metal.md`: dark wet violet/cyan material direction for anatomical/spine-like objects.
- `space-dust-field.md`: layered star dust, nebula, and constellation scene fill.
- `visual-qa-pipeline.md`: browser screenshot QA for real canvas output.
- `agent-meta-learning.md`: strategy selection and episode logging loop.
- `agent-operating-loop.md`: compact start-work-verify-log loop for non-trivial tasks.
- `github-huggingface-research-loop.md`: safe loop for learning from GitHub and Hugging Face without importing clutter.
