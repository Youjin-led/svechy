# Permission Control

## Разрешено без отдельного подтверждения

- Читать и редактировать файлы внутри текущего workspace.
- Добавлять записи в `AGENT_MEMORY.md`, `DECISIONS.md`, `EXPERIMENTS.md`, `TASKS.md`.
- Писать в локальную SQLite-память `.agent_memory.sqlite3`.
- Запускать быстрые локальные проверки: `npm run agent:status`, `npm run agent:context`, `npm run memory:db -- stats`.
- Запускать `npm run qa`, если задача касается frontend/3D-поверхности.

## Требует осторожности или подтверждения

- Долгие генерации Blender/3D-сцен.
- Команды, которые могут менять много файлов.
- Сетевые установки зависимостей.
- Доступ к файлам вне текущего workspace.
- Удаление файлов, очистка директорий, reset/checkout в git.

## Приватность

- Не записывать секреты, токены, пароли и личные данные в память.
- Если в логе встречается приватный путь, оставлять только необходимую техническую часть.
- Для пользовательских предпочтений записывать только то, что помогает работе агента.
