# Презентация проекта Casino Infinity Mirror

Открыть:
- index.html - презентация в браузере.
- casino_mirror_presentation.pdf - PDF для отправки.

Презентация собрана из:
- ТЗ для производства;
- чертежей-пояснений для конструктора;
- 3D-модели для конструктора.

Важно: презентация не заменяет рабочую КД. Она нужна для объяснения проекта и передачи задачи конструктору/цеху.
