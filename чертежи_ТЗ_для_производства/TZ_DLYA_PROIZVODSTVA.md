# Техническое задание для конструктора/производства

Проект: зеркальная casino infinity-композиция с MDF-декором и LED-подсветкой  
Дата подготовки: 11.06.2026  
Статус документа: предпроектное ТЗ, не финальная КД

## 1. Цель

Разработать рабочую конструкторскую документацию и изготовить крупную декоративную зеркальную композицию по референсу: casino/infinity mirror, золотистая световая рамка, игральные карты, покерные фишки, масти, глубокое отражение за счет зеркального короба.

## 2. Исходные габариты

| Параметр | Значение |
|---|---:|
| Наружная ширина | 2125-2130 мм |
| Наружная высота | 1850-1855 мм |
| Наружная глубина | 235 мм |
| Расчетный габарит для предпроекта | 2130 x 1850 x 235 мм |

Допуск исходных габаритов для проектирования: +/- 5 мм.  
Финальные размеры зеркал выдавать только после разработки конструкции короба и посадочных пазов.

## 3. Материалы по заданию заказчика

| Узел | Материал |
|---|---|
| Передний слой | полупрозрачное зеркало / one-way mirror |
| Задний слой | обычное зеркало |
| Боковые внутренние стенки | зеркало |
| Внутренние декоративные объекты | MDF, вырезка + ручная роспись |
| Свет | теплая/янтарная LED-подсветка |
| Корпус | решает конструктор: металл/фанера/МДФ/комбинированная силовая рама |

## 4. Описание эффекта

Изделие должно работать как зеркальный короб с эффектом бесконечного отражения. Реальные объекты устанавливаются один раз на внутренних слоях, а повторение в глубину создается отражениями между передним полупрозрачным зеркалом, задним зеркалом и боковыми зеркалами.

Физически не нужно делать все повторяющиеся карты/фишки, видимые на референсе в глубине. Они должны появляться за счет отражения.

## 7. Предварительная раскладка по глубине

Глубина изделия: 235 мм.

| Слой от зрителя | Глубина | Состав |
|---|---:|---|
| 0 | 0 мм | переднее полупрозрачное зеркало |
| 1 | 20-35 мм | передний LED-контур, мелкие масти |
| 2 | 55-85 мм | фишки и жетоны |
| 3 | 90-125 мм | крупный веер карт |
| 4 | 150-190 мм | задний декор/дополнительный свет |
| 5 | 220-235 мм | заднее зеркало |

Конструктор должен определить реальные дистанционные втулки, крепления, доступ к проводке и допустимые зазоры.

## 8. Зеркала

Предварительная ведомость зеркал:

| Код | Деталь | Кол-во | Предварительный размер |
|---|---|---:|---:|
| M01 | переднее полупрозрачное зеркало | 1 | 2110 x 1830 мм |
| M02 | заднее зеркало | 1 | 1950 x 1670 мм |
| M03-M04 | боковые зеркала | 2 | 1670 x 235 мм |
| M05-M06 | верхнее/нижнее зеркало | 2 | 1950 x 235 мм |
| M07 | внутренние зеркальные полосы/контуры | по проекту | 20-45 мм ширина |

Эти размеры не отдавать в резку без разработки короба. Нужны посадочные пазы, припуски, способ крепления, обработка кромки и допуск на монтаж.

## 9. LED-подсветка

Ожидаемый эффект: теплое янтарно-золотое свечение, как на референсе.

Предварительная длина:

| Зона | Длина |
|---|---:|
| Наружный световой контур | около 7.6 м |
| Внутренний световой контур | около 6.1 м |
| Центральный контур тоннеля | около 4.5 м |
| Вертикальные/декоративные линии | 5-8 м |
| Итого с запасом | 23-28 м |

Рекомендация к проектированию: 24V, 2-3 зоны питания, запас блока питания 25-30%, сервисный доступ к контроллеру и БП.

## 10. Требования к рабочей документации

Конструктор/производство должны подготовить:

1. Чертеж общего вида.
2. Разрез по глубине.
3. Узлы крепления переднего зеркала.
4. Узлы крепления заднего зеркала.
5. Узлы крепления боковых зеркал.
6. Конструкцию силового короба.
7. Способ подвеса или установки.
8. Расчет веса.
9. Схему доступа к LED и блоку питания.
10. Финальный раскрой зеркал.
11. Финальный раскрой MDF.
12. Финальные DXF/SVG-контуры всех художественных деталей.
13. Спецификацию крепежа, проводки и расходников.

## 11. Что нельзя делать без доработки

- Нельзя резать зеркала по предварительным размерам из этого ТЗ.
- Нельзя считать визуальный фасад рабочим чертежом.
- Нельзя клеить тяжелые зеркала только на силикон без механического удержания.
- Нельзя закрывать блок питания без сервисного доступа.
- Нельзя финализировать подвес без расчета веса.

## 12. Ожидаемый результат от конструктора

Рабочий комплект КД/чертежей, по которому производство сможет:

- изготовить короб;
- заказать зеркала;
- вырезать MDF-декор;
- расписать детали;
- установить LED;
- собрать изделие;
- безопасно доставить и смонтировать объект.

Финальная КД должна быть утверждена заказчиком до закупки дорогих зеркал и до резки MDF.
