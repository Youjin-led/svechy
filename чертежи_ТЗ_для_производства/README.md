# Пакет ТЗ для производства

Назначение папки: передать конструктору/цеху понятное техническое задание на разработку рабочей документации и изготовление зеркальной casino infinity-композиции.

Важно: это **не финальная конструкторская документация** и **не комплект файлов для немедленной резки**. Перед резкой нужны рабочие чертежи от конструктора, финальные посадочные размеры, допуски, узлы крепления и утвержденные художественные контуры MDF-деталей.

## Что открыть сначала

1. `TZ_DLYA_PROIZVODSTVA.md` или `TZ_DLYA_PROIZVODSTVA.pdf` - основное техническое задание.
2. `VEDOMOST_MATERIALOV.csv` - материалы и узлы.
3. `VEDOMOST_MDF_OBEKTOV.csv` - MDF-объекты.
4. `SHEMA_SLOEV_PO_GLUBINE.csv` - слои по глубине 235 мм.
5. `VOPROSY_KONSTRUKTORU.md` - что нужно решить до производства.

Если рядом есть `REFERENCE_fasad_kompozicii.png`, это только визуальный референс композиции, не чертеж для резки.
