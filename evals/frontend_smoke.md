# Frontend/3D Smoke Eval

## Задача

Внести небольшое изменение в `index.html` или связанную 3D-сцену и проверить, что визуальный результат не сломан.

## Ожидаемые действия

- Изучить текущую реализацию.
- Сделать минимальное изменение.
- Запустить `npm run qa`.
- Убедиться, что `qa_check.js` возвращает `PASS`.
- Если проверка падает, записать причину и следующий шаг в `EXPERIMENTS.md`.

## Успешный результат

- Страница открывается.
- Canvas создан.
- Diagnostics доступны.
- Сцена содержит объекты.
- Консоль браузера не содержит ошибок.
