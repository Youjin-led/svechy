# Список необходимых ресурсов

## Обязательные

### Домен

Рекомендуемый формат:

- `aimonitorstudio.ru`
- `ai-monitoring.ru`
- `ai-analyst.ru`
- `monitor-ai.ru`

Где купить:

- REG.RU;
- Timeweb;
- Ru-Center.

### Хостинг/VPS

Для старта:

- 1 CPU;
- 1-2 GB RAM;
- 20-40 GB SSD;
- Ubuntu;
- Node.js;
- SSL.

Провайдеры:

- Timeweb Cloud;
- REG.Cloud;
- RuVDS;
- Selectel.

### API поиска

Старт:

- текущий HTML-поиск для демо;
- потом Brave Search API.

### AI API

Варианты:

- OpenAI API;
- OpenRouter как прослойка для разных моделей;
- DeepSeek API, если нужен дешевый режим;
- локальные модели позже, если появятся требования по данным.

### Коммуникации

- Telegram-канал;
- Telegram для заявок;
- VK-сообщество;
- email на домене.

### Продажи

- Авито;
- ручные сообщения;
- личные контакты;
- партнеры: маркетологи, веб-студии, CRM-интеграторы.

## Документы

Нужно подготовить:

- КП;
- бриф;
- договор или простой счет/акт;
- политика конфиденциальности для сайта;
- пользовательское соглашение, если будет личный кабинет;
- шаблон акта выполненных работ.

## Технические ресурсы

- Git-репозиторий;
- резервные копии;
- таблица клиентов;
- таблица расходов;
- журнал инцидентов;
- база знаний по внедрениям.
