# Client Monitoring App

Клиентская демо-версия продукта AI Monitor Studio.

## Назначение

Это не публичный сайт продаж и не внутренний штаб. Это экран продукта, который можно показывать клиенту после заявки:

- список источников мониторинга;
- запуск анализа;
- отчет с сигналами, рисками и рекомендациями;
- мини-база знаний клиента;
- AI-консультант по отчету и контексту;
- копирование отчета в Markdown.

## Запуск

Открой `index.html` в браузере. Сборка и сервер не нужны.

Данные сохраняются локально в браузере через `localStorage`, чтобы демо можно было настраивать под конкретного клиента перед встречей.
