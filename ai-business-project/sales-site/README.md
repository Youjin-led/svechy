# Public sales site

Публичный сайт продаж AI Monitor Studio.

## Что это

Это чистая страница для клиента. Ее можно размещать на домене и вести на нее трафик из Авито, Telegram, VK и личных сообщений.

## Перед публикацией

Заменить:

- `hello@example.com` на реальную почту;
- ссылку Telegram в футере;
- `https://example.com` в `robots.txt` и `sitemap.xml`;
- при необходимости добавить политику конфиденциальности.

## Как открыть

Открыть `index.html` в браузере.
