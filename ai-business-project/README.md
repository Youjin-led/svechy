# AI Operations Studio

Бизнес-проект по внедрению ИИ в малый и средний бизнес.

## Что внутри

- `index.html` - презентационная страница проекта для клиента или партнера.
- `styles.css` - визуальная система страницы.
- `app.js` - интерактивный калькулятор эффекта и переключатели пакетов.
- `docs/BUSINESS_PROJECT.md` - полный бизнес-план.
- `docs/IMPLEMENTATION_PLAYBOOK.md` - методика внедрения ИИ в компанию клиента.
- `docs/SALES_PLAYBOOK.md` - офферы, скрипты и воронка продаж.
- `docs/FINANCIAL_MODEL.csv` - базовая финансовая модель на 12 месяцев.
- `docs/RISKS_AND_GOVERNANCE.md` - риски, безопасность и правила ответственного ИИ.
- `research-assistant.html` - MVP пилота для сбора информации из интернета и аналитического отчета.
- `research-server.js` - локальный сервер с реальным поиском, загрузкой страниц и первичным анализом источников.
- `ai-monitoring-offer.html` - рыночная упаковка продукта "AI-мониторинг ресурсов под ключ".
- `outreach-dashboard.html` - рабочий штаб первых продаж: тексты, лиды, статусы и CSV-экспорт.
- `service-catalog.html` - каталог AI-пакетов под разные бизнес-задачи.
- `company-launch-map.html` - визуальная дорожная карта запуска компании.
- `sales-site/` - чистый публичный сайт продаж для домена и трафика.
- `client-monitoring-app/` - клиентская демо-версия продукта: источники, отчет, рекомендации, мини-база знаний и копирование отчета.
- `docs/RESEARCH_ANALYST_PLAYBOOK.md` - методика внедрения интернет-аналитика.
- `docs/COMMERCIAL_PROPOSAL_AI_MONITORING.md` - коммерческое предложение.
- `docs/DEMO_REPORT_AI_MONITORING.md` - пример отчета для демонстрации.
- `docs/SALES_SCRIPT_AI_MONITORING.md` - скрипт продажи.
- `docs/OBJECTIONS_AI_MONITORING.md` - борьба с возражениями.
- `docs/MEETING_DEMO_SCENARIO.md` - сценарий демонстрации клиенту.
- `docs/CLIENT_BRIEF_AI_MONITORING.md` - бриф для запуска пилота.
- `docs/GO_TO_MARKET_AI_MONITORING.md` - где размещаться и как получать первых клиентов.
- `docs/FIRST_OUTREACH_ECOMMERCE.md` - объявления и личные сообщения для e-commerce.
- `docs/ECOMMERCE_DEMO_SCENARIO.md` - демо-сценарий для e-commerce.
- `docs/CONTENT_PLAN_ECOMMERCE.md` - контент-план на 14 дней.

## Позиционирование

AI Operations Studio помогает компаниям внедрять ИИ там, где он быстро дает деньги:
продажи, поддержка, документы, маркетинг, закупки, аналитика и операционные процессы.

Главный принцип проекта: не продавать "доступ к нейросети", а внедрять измеримый бизнес-процесс с KPI, регламентами, обучением команды и сопровождением.

## Быстрый запуск

Открой `index.html` в браузере. Страница не требует сборки и новых библиотек.

Для рабочего интернет-аналитика запусти:

```bash
node ai-business-project/research-server.js
```

Затем открой:

```text
http://127.0.0.1:5190/research-assistant.html
```

## Первые действия

1. Выбрать нишу для первых продаж: услуги, торговля, недвижимость, производство или образование.
2. Провести 10 коротких интервью с владельцами бизнеса.
3. Продать 2 пилота "AI-аудит + автоматизация одного процесса за 14 дней".
4. Собрать кейсы с цифрами: экономия часов, скорость обработки заявок, прирост конверсии.
5. Упаковать 3 тарифа и перейти к ежемесячному сопровождению.
